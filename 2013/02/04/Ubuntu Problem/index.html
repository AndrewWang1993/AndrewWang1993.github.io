Reading package lists… Error

This problem occurs when I open an ssh link to VPS, and the direct influence is I can not update system and install software through “apt-get **” and “apt-get update/upgrade”, Referring some advice from google such as “ apt-get clean “, “rm -rf /var/cache/apt/.bin” etc… doesn’t fix the problem, Suddenly I realize It could be low memory problem because my VPS only have 786MB memory, so I run “service apache2 restart” && “apt-get update” to free some memory, the error is disappear.

Notice: you can also use other method to free some temporary memory other than terminal Apache server.
